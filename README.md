# Simple Interest Calculator

This project calculates Simple Interest using the formula:

SI = (P × R × T) / 100

Where:
- P = Principal Amount
- R = Rate of Interest
- T = Time Period

## Usage

Enter:
- Principal Amount
- Rate of Interest
- Time Period

The program will calculate and display the Simple Interest.

## Formula

Simple Interest = (Principal × Rate × Time) / 100

## Example

Principal = 1000
Rate = 5
Time = 2 years

Simple Interest = (1000 × 5 × 2) / 100 = 100
